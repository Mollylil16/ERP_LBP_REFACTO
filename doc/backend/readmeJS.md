# Guide JavaScript

Le JavaScript de l’application est stocké dans public/assets/js/app.js.

## Rôle du fichier JS

- interactions simples sur les pages
- gestion des messages flash
- petits effets d’interface
- logique front légère

## Comment modifier le JS

1. ouvrir public/assets/js/app.js
2. repérer la fonction ou l’événement à modifier
3. tester dans le navigateur après chaque changement

## Bonnes pratiques

- garder le JS simple et lisible
- éviter de mélanger logique métier et interface
- utiliser les scripts existants plutôt que d’en ajouter trop

## Astuce

Si vous ne savez pas où ajouter du code, commencez par les petites interactions visibles sur la page.
