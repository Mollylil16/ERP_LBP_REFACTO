# Guide CSS

Les styles de l’application sont centralisés dans public/assets/css/app.css.

## Où modifier le style

- app.css : styles principaux de l’interface
- views/layouts/ : structure globale des pages
- views/ : contenu des pages et composants spécifiques

## Structure du CSS

- styles de base : couleurs, polices, marges
- styles de connexion : page login
- styles du dashboard : portail, cartes, modules
- styles responsive : adaptation mobile

## Conseils

- utiliser les classes déjà existantes quand c’est possible
- ajouter des couleurs et marges cohérentes avec le design actuel
- tester la page sur mobile et desktop

## Astuce pour débuter

Commencez par modifier les classes les plus visibles : boutons, cartes, backgrounds et titres.
